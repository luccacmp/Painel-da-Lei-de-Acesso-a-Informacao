# Painel-da-Lei-de-Acesso-Informa-o
Por meio do Painel da Lei de Acesso à Informação é possível extrair dados sobre os pedidos de informações encaminhados ao Ministério da Fazenda, tais como: número de pedidos respondidos pelo Serviço de Informação ao Cidadão, tempo médio de resposta, assuntos mais demandados, recursos, dentre outras.
