# Painel da Lei de Acesso à Informação

Este repositório apresenta o **Painel da Lei de Acesso à Informação**, desenvolvido em **Power BI**, com o objetivo de disponibilizar de forma visual e interativa os principais dados relacionados aos pedidos de informação encaminhados ao Ministério da Fazenda.

## Acesso ao painel
[Acessar painel público](https://app.powerbi.com/view?r=eyJrIjoiOWY1ZWM4ZjgtYjFmNy00NGVjLWIzOWEtZWQyMzUwNzNkYWMyIiwidCI6IjQ1NjIxM2NmLTcwNzMtNDdjNi1iZjQ5LTQxYjY1NGFkNDQ5YiJ9)

## Sobre o projeto
Por meio do Painel da Lei de Acesso à Informação, é possível consultar dados sobre os pedidos de informação encaminhados ao Ministério da Fazenda, como número de pedidos respondidos pelo Serviço de Informação ao Cidadão, tempo médio de resposta, assuntos mais demandados, recursos, entre outros indicadores relevantes.

## Tecnologias utilizadas
- Power BI
- Power Query
- DAX
- Modelagem e visualização de dados
- Excel
